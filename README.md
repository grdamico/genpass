# genpass

## 📘 Overview

`genpass` is a simple command-line tool to generate random passwords.

It was created as a small utility project to practice basic programming concepts such as randomness, string manipulation, and command-line interaction.

---

## 🚀 Features

* Generate random passwords
* Configurable length
* Simple and lightweight
* Easy to use from the command line

---

## 🛠️ Usage

Run the program:

```bash
genpass
```

Example output:

```
a8fK2pLm
```

You can customize the password length (if supported):

```bash
genpass 12
```

---

## 🎯 Purpose

This project is intended for:

* Learning and experimentation
* Practicing basic programming skills
* Understanding how password generators work

It is **not designed for production use or security-critical environments**.
